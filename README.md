#BMI Calculator (Python + PyWebIO)

This repository contains a simple **BMI (Body Mass Index) Calculator** implemented using **Python** and **PyWebIO**.  
The program takes user input for height and weight and calculates BMI along with the corresponding health category.

---

#BMI Formula
BMI = Mass (kg) / (Height (m))Â²



Where:
- Mass is measured in Kilograms (kg)
- Height is measured in Meters (m)

---

#Features

- Web-based input using PyWebIO
- Accurate BMI calculation
- Automatic BMI classification:
  - Severely underweight
  - Underweight
  - Normal
  - Overweight
  - Moderately obese
  - Severely obese
- Beginner-friendly Python code
